## Quick Start

1. Clone the repo
  ```
  $ git clone https://github.com/app-generator/flask-learn-by-coding.git
  $ cd flask-learn-by-coding/src
  ```

2. Install the dependencies:
  ```
  $ pip install -r requirements.txt
  ```

3. Setup the environment:
  ```
  $ export FLASK_APP=run.py
  ```

4. Run the app:
  ```
  $ flask run
  ```
